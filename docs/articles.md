# 文章推荐

- [秋葉aaaki - AI绘画知识合集](https://www.bilibili.com/read/readlist/rl670536)
  - 这个网页是B站UP主"秋葉aaaki"的"AI绘画知识合集"专栏文集，共包含6篇专栏文章。这些文章系统性地介绍了AI绘画（特指Stable Diffusion）的入门教程、各类模型的种类与使用方法、AI绘画的工作原理、相关论文资源，并纠正了关于VAE（变分自编码器）和模型大小的常见误区，旨在为从零开始的初学者到希望深入研究的使用者提供全面、实用的知识指导与参考资料。
- [细节修复（Detailer）原理与操作](https://www.bilibili.com/opus/1037542722974515205)
  - 该网页主要介绍了AI绘画中**细节修复（Detailer）** 的原理与操作方法。作者指出，细节崩坏的根本原因是画面中面部、手部等区域分配的**像素不足**，单纯拉高整体分辨率并不可行。为此，文章提供了两种解决方案：一是**潜空间放大**，在解码前对潜空间数据进行放大，以提升整体质感；二是**使用Detailer节点**，即单独抠出需要修复的部位（如脸、手），以更高分辨率进行图生图，再将修复后的部分贴回原图。文章详细解读了Detailer中 `guide_size`、`max_size` 等关键参数的作用，并强调AI能力有限，最终仍需结合手动修图来获得理想效果。
